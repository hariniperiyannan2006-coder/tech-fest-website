# College Tech Fest 2026 Website

## Overview

This is a simple event website created using HTML. The website provides information about a college technical festival, including event details, registration, and contact information.

## Features

* Home page with event information
* Event details and guidelines
* Registration form for participants
* Registration success page
* Contact information section
* Images related to the Tech Fest

## Technologies Used

* HTML5

## Project Structure

* `index.html` – Home page
* `register.html` – Registration form
* `success.html` – Registration success page
* `img/` – Images used in the website

## Purpose

This project was created to practice web development fundamentals and learn how to build a simple multi-page website using HTML.
